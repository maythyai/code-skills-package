**CSP — top commands**

```text
/csp-new-project           Initialize a project (greenfield)
/csp-map-codebase          Map an existing codebase (brownfield)
/csp-plan-phase <N>        Create a phase plan
/csp-execute-phase <N>     Execute a phase
/csp-progress              Where am I, what's next
/csp-quick                 Small ad-hoc task with CSP guarantees
/csp-fast "<task>"         Trivial inline task — no subagents
/csp-debug "<symptom>"     Persistent debug session (survives /clear)
/csp-capture               Save an idea / todo / note
/csp-ship <N>              Open a PR from a completed phase
```

More: `/csp-help` (default tour) · `/csp-help --full` (everything) · `/csp-help <topic>` (one section)
