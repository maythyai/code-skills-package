# Spec, ADR, Implementation Plan & PRD Templates

Skeletons for the four artifacts generated by `csp-spec-adr`. Copy the relevant
block, fill every placeholder, and delete sections that genuinely do not apply
(never leave placeholder text in a shipped document).

---

## A. Specification Template

Save as `/spec/spec-[purpose]-[slug].md`.

```markdown
---
title: [Concise Title]
version: 1.0
date_created: YYYY-MM-DD
last_updated: YYYY-MM-DD
owner: [Team / Individual]
tags: [infrastructure, process, design, ...]
---

# Introduction

[One paragraph: what this spec covers and the goal it serves.]

## 1. Purpose & Scope

What this spec governs, who it is for, and what is explicitly out of scope.

## 2. Definitions

| Term / Acronym | Meaning                                      |
|----------------|----------------------------------------------|
| API            | Application Programming Interface            |
| ...            | ...                                          |

## 3. Requirements, Constraints & Guidelines

- **REQ-001**: [Requirement]
- **SEC-001**: [Security requirement]
- **CON-001**: [Constraint]
- **GUD-001**: [Guideline]
- **PAT-001**: [Pattern to follow]

## 4. Interfaces & Data Contracts

Describe APIs, schemas, events, and integration points. Use tables or
code blocks for payload shapes.

```json
{ "eventId": "uuid", "type": "user.created", "payload": { "userId": "string" } }
```

## 5. Acceptance Criteria

- **AC-001**: Given [context], When [action], Then [outcome].
- **AC-002**: The system shall [behavior] when [condition].

## 6. Test Automation Strategy

- **Frameworks**: [pytest / Jest / JUnit / ...].

## 7. Rationale & Context

Why the requirements and constraints above were chosen.

## 8. Dependencies & External Integrations

- **EXT-001**: [External system] — [purpose].

- **INF-001**: [Infrastructure] — [constraints].

## 9. Examples & Edge Cases

## 10. Validation Criteria

## 11. Related Specifications / Further Reading

- [External documentation](https://...)

  - KPI-3: [measurable target].
