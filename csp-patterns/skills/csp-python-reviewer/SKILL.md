# Python Reviewer

Python code review checklist:
1. PEP 8 compliance
2. Type hints present and correct
3. Pythonic idioms (list comprehensions, context managers)
4. Security considerations (input validation, no eval())
5. Performance patterns (avoid unnecessary copies)
6. Docstrings complete
7. Error handling appropriate
